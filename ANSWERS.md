- [ ] Why would you use class component over function components (removing hooks from the question)?
    If the company that you were working at were using class components. Honestly, they’re more of a hassle than using function components and hooks.
- [ ] Name three lifecycle methods and their purposes.
    Mount—the birth of the the state of the app that renders
	Update—when there is an update, it’s the growth and change of life
	Unmount—for when the rendering is complete (or to save space in in like a video game) it will complete and ‘die’ so to speak when you are finished with that data
- [ ] What is the purpose of a custom hook?
    a custom hook is so that you can do multiple functions and hooks at once—this is also so that you can have more DRY code
- [ ] Why is it important to test our apps?
    so that we don’t have to do it manually, which would take a lot of time and money. It also helps you write better code to get into practice for writing these tests.
